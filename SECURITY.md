# Security Policy
SECUIRITY POLICY IS TO BE UPDATED AFTER NEW UPDATES
## Reporting a Vulnerability
report bugs in public issues.
report an issue privately ONLY if it involves security leaks.

### how to report
simply:

log or screenshot the events

email them to unsodou@gmail.com

They'll be looked at within a maximum of 2 days, and we'll fix it and send you a response email.
